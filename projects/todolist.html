To-Do List

A to-do list app // EJS & MongoDB

Description

This is a totally functional Node app that uses a MongoDB database to create a to-do list app.  The front-end is created using HTML and CSS. For the back-end I used EJS for templating the different custom lists and a database with MongoDB to store the lists data.  Ticking the check boxes will delete the item from the list and the database, so all your lists are saved and you can come back to them at a later time.  You can create custom lists by adding a path to the URL (i.e.: www.todolist.borrega.io/shopping).  This website is mobile-friendly. 

Lessons learned

The implementation of a database was the more intricated part of this project, but using MongoDB really helped with its development and made it fully-functional for anyone to use. 

Next steps

Extra functionality could be added, such as displays of the different custom lists on the main page, with the capability to remove them completely from the database.  Customisation with colours and buttons for the different lists would also be possible to make the app more interactive.
