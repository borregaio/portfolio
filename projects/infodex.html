Infodex

Information about Pokémon // API

Description

I have always been a Pokémon fan since the very beginning. What would be cooler than having your own Pokédex?  Infodex is a Node web app that uses an open API to fetch every Pokémon information such as name, type, number and image.  I created the front-end design with CSS and added some functionality with JavaScript to remove the first div after a user has searched for a Pokémon to reveal the information.  This website is mobile-friendly. 

Lessons learned

Using the “fetch()” function was a much easier way to get all the information needed for this app.  Implementing JavaScript to remove and show divs when searching for a specific Pokémon made the code simple and I was able to keep it in just one file. 

Next steps

A good way of expanding this project would be featuring more information about each Pokémon such as stats, evolutions and attacks.  Expanding the app with some form of templating, like EJS, would give more flexibility to each Infodex entry and it would allow the code to be refactored in a cleaner way.
