Melodi

A landing page for a music streaming app // HTML & CSS

Description

Landing pages are extremely useful to new business or start-ups that need to feature a product and be commercial.  I created Melodi, a landing page for a music streaming app. This page is a clean and straightforward site where you can learn features about the app, read user’s testimonials and check the different price options.  This page is a mock-up where I mainly implemented HTML, CSS and Bootstrap. With the later one, it helped me to achieve that clean and professional look with button animations and carousel implementation that give the website a modern and professional look.  The website is mobile-friendly. 

Lessons learned

As this was one of my first web projects, it really helped me to understand how a business website should look and what functionalities should have. I learned how to display the information in a clear and intuitive way so the user finds the website accessible. 

Next steps

If I had to work more on this project, my next steps would be giving each section their own page, developing more information about them and implementing some database with Mongo DB to sign up new users.
