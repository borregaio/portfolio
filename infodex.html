01. Infodex

NodeJS & RESTful API

Project Website GitHub Page

The Goal

I have always been an avid Pokémon fan, and having a personal Pokédex is incredibly cool. Infodex, a Node web app, utilises an open API to fetch comprehensive Pokémon information, including name, type, number, and image. The front-end design, created with CSS, incorporates JavaScript functionality to remove the initial div after a Pokémon search, revealing the desired information. Additionally, this website is mobile-friendly.

The Challenges

The utilisation of the "fetch()" function proved to be a significantly more convenient approach for acquiring the entirety of the information required by this application. By implementing JavaScript to selectively remove and display divs in response to specific Pokémon search queries, the code was rendered concise and efficiently organised within a single file.

What's Next?

A viable approach to expanding this project would involve incorporating additional details regarding each Pokémon, encompassing information such as statistics, evolutions, and attacks. Augmenting the application with a templating system, such as EJS, would provide increased versatility to individual Infodex entries, while facilitating a more streamlined and refined code refactoring process. This integration would enable the codebase to be structured in a more organised and efficient manner, enhancing both the functionality and maintainability of the application.
