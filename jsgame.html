02. Rock, Paper Scissors

JavaScript Game

Project Website GitHub Page

The Goal

Welcome to "Rock, Paper, Scissors" - a JavaScript game where you choose rock, paper, or scissors, and the computer picks randomly. The result is displayed on screen, and scores are updated. Enjoy the challenge and have fun!

The Challenges

Creating the "Rock, Paper, Scissors" game posed challenges in breaking down processes into functions. I used an array for selections, added event listeners for player's choice, randomized computer's choice, displayed both selections, and updated the score. Learning the logic and implementing different functions was tough, but I discovered dataset and parseInt() for added knowledge. I'm looking forward to exploring more complex games with JavaScript!

What's Next?

This game serves as a simple demonstration of various JavaScript functions and DOM manipulation. It showcases the possibilities of building interactive experiences. My next endeavor will involve creating a more complex game, employing different logic, and implementing a wider range of JavaScript functions to elevate the gaming experience further.
