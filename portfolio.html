03. Portfolio

NodeJS & EJS

Project Website GitHub Page

The Goal

Benjamin Wigmore, a textile designer, desired a simple portfolio website that showcased his work without personal information, resembling a concise digital business card. The focus was to highlight his artistic talent and craftsmanship, ensuring a minimalist user experience that captivated viewers with captivating textile designs.

The Challenges

EJS templates were utilized for consistent project layouts, alongside a persistent sidebar. Balancing the background video size ensured quick loading without compromising quality. JavaScript modals were implemented for image zooming, while Bootstrap facilitated the creation of a dropdown menu for the "Projects" section. These solutions improved functionality and user experience.

What's Next?

In the future, expanding Ben's portfolio could involve adding sections like an "About Me" and providing more information on each project, including the creative process and concept behind them. Additionally, upgrading the hosting service would improve the website's loading speed, resulting in a better user experience.
